Module End Exam:

Q. Create an MVC Application for managing orders named "OrdersMVC". Create the following things in the application.
  a. Create a controller name OrdersController
  b. Create the following actions(GET & POST)
      a. AddOrder (for adding an order in database and show the auto generated orderid)
      b. SearchOrder (Show the orders from database based on orderid or customername)
      c. Index (for showing the links: Add Order & SearchOrders)
  c. Models:
      a. OrderModel(OrderID, CustomerName, ItemName, ItemPrice, ItemQty) with validation
      b. OrderDAL: (Create Two Functions AddOrder and SearchOrders)
  d. View: 
      AddOrder, ViewOrders, Index
  e. Create a database and a Table named tbl_orders
  f. SQL TABLE:
      tbl_Orders - OrderID(Auto, PK), CustomerName, ItemName, ItemPrice, ItemQty.
